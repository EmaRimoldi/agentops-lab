# Figures

Public figures for the agent memory ablation experiment. They are generated by
`scripts/plot_agent_memory_ablation.py` from `../trial_results.json`.

All figures use the gap-free public trial index `T01`-`T11`. Corrupted or
non-executed historical cells are not plotted.

| file | message |
| --- | --- |
| `figure-01-trial-outcomes.png` / `.pdf` | best validation BPB reached by each valid trial |
| `figure-02-memory-stabilization.png` / `.pdf` | shared memory stabilizes exploratory agents compared with no-memory controls |
| `figure-03-training-attempts.png` / `.pdf` | number of training attempts produced by each valid trial |
