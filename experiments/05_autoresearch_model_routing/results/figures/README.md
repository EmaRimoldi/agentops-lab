# Figure Inventory

This directory preserves figure files imported from the `NeurIPS_2026`
AutoResearch campaign. Prefer the validation-loss, success, tau, improvement,
and router diagnostics for reader-facing material. Figures centered on
composite deployment-cost accounting are retained for provenance but are not
recommended for presentation.

## Recommended Figures

- `threeworker_improvement_distribution.{pdf,png}`: distribution of final
  validation-loss improvements by worker.
- `threeworker_relative_improvement_trajectories.{pdf,png}`: improvement
  trajectories over proposal steps.
- `threeworker_tau_distribution.{pdf,png}`: first successful step distribution.
- `threeworker_threshold_sensitivity.{pdf,png}`: sensitivity to the success
  threshold.
- `threeworker_router_paired_gain.{pdf,png}`: paired router-gain diagnostic.
- `threeworker_router_selection_regret.{pdf,png}`: router selection regret.
- `threeworker_negative_controls.{pdf,png}`: control checks for router signals.
- `first_hit_ecdf_by_mode.{pdf,png}`: cumulative first-hit curves.
- `quality_vs_certified_resource.{pdf,png}`: quality versus certified resource.
