Claude Code workflow evaluation

Build N agents. Measure whether they beat one.

Agent Workflow lets developers define custom Claude Code agent teams, run them in isolated workspaces, and compare single, parallel, shared-memory, swarm, and merge workflows before scaling them.

View on GitHub Run locally

Quick start

Clone the repo and run the preflight check. Live agent runs require Claude Code authentication.

git clone https://github.com/EmaRimoldi/agent-workflow.git
cd agent-workflow
uv run agent-workflow doctor

What it gives you

Configurable rosters

Define one agent or N agents with explicit roles, models, memory modes, and device assignment.

Comparable runs

Use fixed-step evaluation so hardware contention does not masquerade as model quality.

Evidence bundles

Preserve configs, logs, trajectories, snapshots, metrics, and reports for later review.
